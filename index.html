HTML and Web Basics

The World Wide Web is a collection of HTML documents. HTML stands for hypertext markup language; it is the code behind HTML documents that tells computers and browsers how to display information. HTTP stands for hypertext transfer protocol; it defines the rules for transferring files among computers and software (like browsers) on the world wide web.

HTML is composed of the following:
  • text
  • markup
  • references to other documents (like vidoes, images, audio files, etc)
  • links to other documents (i.e. pages)
An element of HTML is composed of:
  • an opening tag
  • content
  • a closing tag (except in the case of void tags)

Tags may have multiple attributes which add or change the functionality of an element.

In general, don't worry about memorizing bits of code; anything you haven't memorized by common use is easily found with a Google search.
